资源下载

1.不打开 Web 页或图片而直接保存

用鼠标右键单击目标项目的链接，然后单击“目标另存为”。 浏览器中就会弹出窗口,选择所需存放的目标目录,点击保存,浏览器将会显示下载文件的进度直到完成下载。

2.在文件下载中应注意的问题

某些文件可能携带病毒或危害您的计算机。
如果现在运行该程序或打开该文件，可能会导致计算机或其他程序关闭，从而丢失计算机上打开的文件中尚未保存的工 作。
如果将该文件或程序保存到磁盘，可以在以后运行此文件或程序。在打开文件或运行程序之前，必须注意以下事宜：

1.使用查病毒程序检查文件。
2.保存工作并关闭其他程序。
3.与 Internet 或任何其他网络断开连接。
4.为安全起见，请确认已选中“打开之前始终询问”复选框。
5.如果确信直接从 Internet 打开或运行此类文件或程序始终 是安全的，可以清除此复选框。

注意
1. “打开这种类型的文件之前始终询问”复选框对某些文件类型无效，例如扩展名为 .exe 或 .com 运行程序或命令的文 件。如2.果该复选框无效，那么在每次打开此类型的文件之前您都要被询问。
3.保存 Web 页中的图片或文本 浏览 Web 页时，经常会遇到需要保存以便将来参考或与他人共享的信息。
4.可以将 Web 页的全部或部分内容（文本、 图像或链接）保存起来。
5.对于无法访问 Web 或没有计算机的人，您可以将 Web 页打印出来。
